# Soil Moisture Sensor

## Introduction
This is an IoT-based Soil Moisture Sensor project. It measures the moisture level of the soil.

## Objective
- Monitor soil moisture.
- Save water.
- Help farmers irrigate efficiently.

## Components Used
- Arduino Uno
- Soil Moisture Sensor
- Breadboard
- Jumper Wires
- USB Cable

## Working
The sensor detects the soil moisture level. Arduino reads the sensor data and shows whether the soil is wet or dry.

## Advantages
- Saves water
- Easy to use
- Low cost
- Automatic monitoring

## Result
The project successfully measures soil moisture and helps in efficient irrigation.
